Instruções para Exportar/Imprimir:
  1. Verifique se todos os cartões foram gerados.
  2. Use o botão "Exportar para PDF / Imprimir" ou Ctrl+P.
  3. No diálogo de impressão:
    • Destino: "Salvar como PDF".
    • Tamanho do Papel: "A4".
    • Margens: "Nenhuma".
    • Escala: **CRUCIAL: "Padrão" ou "100%"**.
    • Opções: Marque "Imprimir fundos".
